# VK Experts 
![Hi](https://github.com/thetimyr/vk_expertes/blob/main/other/icon.png?raw=true)

Данный скрипт представляет собой готовый одностраничный PHP-виджет, который **визуально повторяет интерфейс приложения** «VK Эксперты» (VK Experts), при этом подтягивает актуальные данные пользователя из официального **API ВКонтакте** в реальном времени.

Скрипт отображает:
*   Имя и фамилию пользователя ВК.
*   Аватараку пользователя.
*   Реальное количество записей на стене пользователя **за всё время**.
*   Реальное количество записей **за сегодня**.
*   Сумму лайков на записях за сегодняшний день (опционально скрыто, но присутствует в коде).
*   Задержку (пинг) ответа от API ВК.
*   Статус серверов ВК на основе задержки.
*   *Поля «Рейтинг» и «Тематика ленты», присутствующие в макете, имитируются вручную (так как эти данные не отдаются официальным API ВКонтакте).*

## 🧰 Требования к серверу

Для корректной работы скрипта ваш веб-сервер должен поддерживать:
*   **PHP версии 7.4 или выше** (рекомендуется 8.0+).
*   Расширение **cURL** (включено в большинстве сборок PHP по умолчанию).
*   Модуль **JSON** (встроен в PHP).

Если вы используете локальный сервер (например, OpenServer, MAMP, XAMPP, WAMP) — убедитесь, что расширение `php_curl` активно в настройках `php.ini`.

## 📥 Установка

1.  Скачайте или создайте файл `index.php` в корневой папке вашего веб-сервера (например, в папке `htdocs` или `www`).
2.  Вставьте в этот файл предоставленный PHP-код.
3.  Перейдите к разделу **Настройка** для внесения персональных данных.

## 🔧 Настройка подключения к ВК

Скрипт использует официальный API ВКонтакте. Для получения данных требуется **Сервисный ключ доступа (Service Token)**.

### 1. Получение Сервисного токена
1.  Зайдите на сайт [VK Developers](https://vk.com/dev) и авторизуйтесь.
2.  Нажмите «Создать приложение» (выберите «Платформа: Веб-сайт» или любой другой пункт, это не принципиально). Назовите приложение как угодно.
3.  После создания приложения перейдите во вкладку **«Настройки»**.
4.  Прокрутите страницу вниз до блока **«Ключи доступа»**.
5.  Нажмите кнопку **«Получить»** напротив строки **«Сервисный ключ доступа»**.
6.  Подтвердите действие кодом из СМС или через приложение.
7.  Скопируйте полученный длинный ключ (начинается обычно с `vk1.a.`).

### 2. Вставка токена в скрипт
Откройте файл `index.php` в текстовом редакторе. Найдите в самом начале строку с переменной `$access_token` и замените заглушку на ваш реальный ключ:

php
// Было
$access_token = 'ВАШ_СЕРВИСНЫЙ_ТОКЕН_ВКОНТАКТЕ';

// Стало (пример)
$access_token = 'vk1.a.abc123def456...';
### 3. Указание ID пользователя
Найдите строку $user_id. Укажите ID профиля ВКонтакте, данные которого вы хотите отображать.

php
// Для пользователя id1
$user_id = '1'; 

// Для пользователя id123456
$user_id = '123456';
## ✏️ Кастомизация дизайна и данных VK Experts
Поскольку данные «Рейтинг» (например, 3,4) и «Лента» (например, Стиль) генерируются алгоритмами стороннего приложения VK Experts, официальное API ВК не может их предоставить.

Если вы знаете эти значения или хотите написать их вручную, найдите следующие переменные в начале скрипта:

php
$manual_rating = '3,4'; // Рейтинг, который отобразится в шапке
$manual_theme = 'Стиль'; // Тематика ленты
Вы можете заменить их на свои значения или, если у вас есть навыки веб-парсинга, доработать скрипт для получения их с фронтенда стороннего сайта VK Experts.

Важное примечание: В коде также присутствует переменная $today_likes, которая считает количество лайков на стене за текущие сутки через API ВК. В текущей версии макета она не выведена на экран. Вы можете дописать её в HTML выводе самостоятельно, добавив в блок .user-stats строку: <br>Лайков за сегодня: <?php echo $today_likes; ?>.

## 🚀 Запуск скрипта
### Способ 1. Локальный сервер (для тестирования)
Если у вас не установлен OpenServer/XAMPP, вы можете запустить встроенный сервер PHP через консоль (Командную строку или Терминал):

Откройте терминал.

Перейдите в папку со скриптом: cd C:\путь\к\папке.

Введите команду:

bash
php -S localhost:8000
Откройте браузер и перейдите по адресу http://localhost:8000/index.php.

### Способ 2. Реальный хостинг
Загрузите файл index.php в корень вашего сайта на хостинге (например, в папку public_html). После загрузки перейдите по адресу https://ваш-сайт.ru/index.php.

## ⚠️ Возможные проблемы и ограничения
Лимиты API. Сервисный ключ ВКонтакте имеет ограничения на количество запросов в сутки для обычных приложений (около 250 000 запросов). Для одного пользователя скрипт делает всего 2-3 вызова, поэтому ограничения не страшны.

Приватность пользователя. Если страница пользователя закрыта полностью настройками приватности, API может вернуть ошибку о доступе. Скрипт выведет базовые данные, но записей за сегодня может не быть.

Отсутствие Рейтинга. Напомним еще раз — настоящий «ВК Эксперт» берет данные из закрытых систем внутреннего анализа. Данный скрипт показывает реальное количество постов и лайков, взятых из API, под стилистику приложения.
